# Лабораторная работа №1 - Сетевое программирование

**Выполнил:** Саид Наваф  
**Группа:** K3341  
**Дата:** 2025

## Описание работы

Данная лабораторная работа посвящена изучению основ сетевого программирования на Python с использованием модуля `socket`. В рамках работы реализованы различные типы клиент-серверных приложений, от простого UDP-сокета до полноценного HTTP-сервера.

## Цели работы

1. Изучить принципы работы сетевых протоколов TCP и UDP
2. Освоить программирование сокетов на языке Python
3. Реализовать клиент-серверные приложения различной сложности
4. Понять основы HTTP протокола и его реализации
5. Изучить многопоточное программирование для работы с множественными подключениями

## Структура работы

Работа состоит из 5 заданий:

### [Задание 1 - UDP Сокет](task1.md)
Реализация простого UDP сервера и клиента для обмена сообщениями.

**Технологии:** UDP, socket

### [Задание 2 - TCP Калькулятор](task2.md)
TCP сервер-калькулятор, выполняющий математические вычисления по формуле (a + b) / h.

**Технологии:** TCP, socket, обработка исключений

### [Задание 3 - HTTP Сервер](task3.md)
Простой HTTP сервер, отдающий статическую HTML страницу.

**Технологии:** HTTP/1.1, socket, HTML

### [Задание 4 - Многопользовательский чат](task4.md)
Многопользовательский чат-сервер с поддержкой множественных одновременных подключений.

**Технологии:** TCP, threading, JSON, broadcast messaging

### [Задание 5 - HTTP Сервер с формами](task5.md)
Полноценный HTTP сервер с поддержкой GET/POST запросов, HTML форм и персистентного хранения данных.

**Технологии:** HTTP/1.1, JSON, HTML forms, file persistence

## Критерии оценивания

- **60% баллов** - Выполнение пунктов 1-4 и однопользовательского чата (без потоков)
- **100% баллов** - Выполнение пунктов 1-5 и многопользовательского чата (с потоками)

## Используемые технологии

- **Python 3.x** - основной язык программирования
- **socket** - модуль для работы с сетевыми соединениями
- **threading** - модуль для многопоточности
- **json** - модуль для работы с JSON данными
- **dataclasses** - для создания структур данных
