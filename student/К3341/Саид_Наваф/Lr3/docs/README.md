# ЛР3 — серверная часть (Django + DRF)

Кратко:
- Проект реализует REST API для управления владельцами, удостоверениями, автомобилями и владением.
- Technologies: Python 3.14, Django 4.2+, Django REST Framework, Djoser, drf-spectacular, mkdocs.

Как запустить:
1. Создать и активировать виртуальное окружение
2. pip install -r requirements.txt
3. python manage.py migrate
4. python populate_lab3_practice3_1.py
5. python manage.py runserver

Документация API: открыть `docs/` (mkdocs) или использовать Swagger UI в работе сервера:
- Swagger: http://127.0.0.1:8000/api/docs/swagger/
